Privacy policy

Last updated:

Summary

  • Data minimization: The bot only processes the data required to perform commands and moderation actions.
  • No sale of data: We do not sell or share personal data with third parties for marketing.
  • Transparency: Configuration and logs aim to be readable and opt-in where possible.

Data we process

  • Discord identifiers: User IDs, Guild IDs, Channel IDs, Role IDs, and Message IDs used to execute bot features.
  • Command content: Slash command names and options you provide to perform the requested action.
  • Server configuration: Settings you apply (e.g., enabled modules, language, role mappings).
  • Moderation records: Action type, target ID, moderator ID, timestamp, and reason (if provided).

Data we do not collect by default

  • Message bodies: We do not store full message content unless a specific feature explicitly requires it and you enable it.
  • Sensitive categories: No biometric, health, or financial data is collected.
  • Tracking: No third-party analytics on the bot itself. The website may use basic server logs.

Retention

Configuration and moderation logs are kept only as long as necessary for bot operation and server accountability. You may request deletion of server-specific configuration data via the support server or GitHub issues, and we will remove it after verifying server ownership.

Legal bases and rights

For users in the EU/EEA, processing is based on legitimate interests of server moderation and community management. You may have rights to access, rectify, or delete personal data, and to object to certain processing. Requests can be made via the contact methods below. We will respond within a reasonable timeframe.

Security

We use access controls and minimize stored data. API tokens and credentials are stored securely and not exposed publicly.

Children

TabouLait is intended for Discord users who meet Discord’s minimum age requirements. We do not knowingly collect data from children under applicable age thresholds.

Third parties

The bot interacts with the Discord API. Discord’s handling of data is governed by their own policies. When optional integrations are enabled (e.g., webhooks or status pages), data shared to those services is subject to their policies.

Contact

For privacy requests, takedowns, or questions, open an issue on the GitHub repository or contact us via the support server.

Changes to this policy

We may update this policy to reflect changes to the bot or legal requirements. The “Last updated” date will be adjusted, and material changes will be announced on the support server.
